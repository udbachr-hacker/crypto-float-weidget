# Crypto Floating Widget (加密货币浮动小组件)

[English](#english) | [中文](#中文)

---

## English

A minimalist, frameless, and always-on-top desktop widget for Windows that displays real-time cryptocurrency prices. Built with Electron.

### Features
- **Minimalist Design**: Small, transparent floating window that doesn't distract from your work.
- **Always on Top**: Keeps track of prices even while you are using other applications.
- **Real-time Updates**: Fetches latest prices from Gate.io API.
- **Dynamic Rotation**: Cycles through your selected cryptocurrency pairs with smooth animations.
- **Customizable**: Add or remove any cryptocurrency pair easily through the settings menu.
- **Easy Installation**: Comes with a portable version and an installer.

### Quick Start
1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/crypto-floating-widget.git
   cd crypto-floating-widget
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the application**:
   ```bash
   npm start
   ```
4. **Build the executable**:
   ```bash
   npm run dist
   ```

### Technologies Used
- Electron
- JavaScript / HTML / CSS
- Gate.io API

## 中文

一个极简、无边框、置顶的 Windows 桌面小组件，实时显示加密货币价格。基于 Electron 开发。

### 功能特点
- **极简设计**: 小型透明悬浮窗，不干扰日常办公。
- **窗口置顶**: 即使在使用其他应用程序时，也能随时查看价格。
- **实时更新**: 从 Gate.io API 获取最新的实时价格。
- **动态轮播**: 自动循环播放你选择的币种，并带有平滑的切换动画。
- **高度自定义**: 可通过设置菜单轻松添加或删除任何币种交易对。
- **安装便捷**: 提供便携版 (Portable) 和安装包 (NSIS)。

### 快速开始
1. **克隆仓库**:
   ```bash
   git clone https://github.com/your-username/crypto-floating-widget.git
   cd crypto-floating-widget
   ```
2. **安装依赖**:
   ```bash
   npm install
   ```
3. **运行程序**:
   ```bash
   npm start
   ```
4. **打包发布**:
   ```bash
   npm run dist
   ```

### 技术栈
- Electron
- JavaScript / HTML / CSS
- Gate.io API
